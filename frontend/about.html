Spider.exe — Encyclopédie interactive dédiée à l’univers Spider-Man

À propos du projet

Spider.exe est un site web dédié à l’univers de Spider-Man, regroupant les personnages emblématiques issus des comics, du cinéma, des séries animées et des jeux vidéo.

Ce projet a été réalisé dans le cadre de la préparation au Titre Professionnel Développeur Web et Web Mobile (TP DWWM). Il s’agit d’une encyclopédie interactive permettant de consulter des fiches détaillées de personnages, leurs apparitions majeures et les actualités liées à l’univers Spider-Man.

L’objectif est de proposer une expérience immersive inspirée de l’esthétique comics, tout en respectant les bonnes pratiques du développement web moderne.

Objectifs pédagogiques

Ce projet permet de mettre en œuvre les compétences suivantes :

  • Maquettage d’interfaces utilisateur (Figma)
  • Développement d’interfaces web dynamiques
  • Mise en place d’une base de données relationnelle
  • Développement d’une API côté serveur
  • Gestion des données via un CRUD complet
  • Sécurisation basique d’une application web

Technologies utilisées

Front-end

  • HTML5
  • CSS3 (Flexbox / Grid)
  • JavaScript (DOM, Fetch API)

Back-end

  • Node.js
  • Base de données MySQL
  • API REST

Outils

  • Git / GitHub
  • Figma (maquettes)
  • Visual Studio Code
