    A propos

    A propos du projet

    Spider.exe est un site web dédié à l’univers de Spider-Man, regroupant les personnages emblématiques issus des comics, du cinéma, des séries animées et des jeux vidéo.

    Le projet

    Ce projet a été réalisé dans le cadre de la préparation au Titre Professionnel Développeur Web et Web Mobile (TP DWWM). Il s’agit d’une encyclopédie interactive permettant de consulter des fiches détaillées de personnages, leurs apparitions majeures et les actualités liées à l’univers Spider-Man. L’objectif est de proposer une expérience immersive inspirée de l’esthétique comics, tout en respectant les bonnes pratiques du développement web moderne.

    Objectifs pedagogiques

    Ce projet permet de mettre en oeuvre les compétences suivantes :

    • - Maquettage d’interfaces utilisateur (Figma)
    • - Développement d’interfaces web dynamiques
    • - Mise en place d’une base de données relationnelle
    • - Développement d’une API côté serveur
    • - Gestion des données via un CRUD complet
    • - Sécurisation basique d’une application web

    Technologies utilisees

    Front-end

    • - HTML5
    • - CSS3 (Flexbox / Grid)
    • - JavaScript (DOM, Fetch API)

    Back-end

    • - Node.js
    • - Base de données MySQL
    • - API REST

    Outils

    • - Git / GitHub
    • - Figma (maquettes)
    • - Visual Studio Code

    A propos du developpeur

    Ce site a été conçu et développé par Godji Remi, dans un cadre pédagogique.

    Il reflète les compétences acquises tout au long de la formation DWWM, ainsi que l’intérêt personnel pour l’univers Spider-Man et le design inspiré des bandes dessinées.
